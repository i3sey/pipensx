pipensx development. They are retained for architectural context, but they are
not a current roadmap and may describe code that has since changed or been
removed.

The source code, tests, [README](../../README.md), and
[build guide](../../BUILD.md) are authoritative. New work should be tracked in
GitHub issues rather than by extending these files.
